#### (No Whitespace before empty braces)

Write ```f()``` and not ```f ()```.

#### (Whitespace before non-empty braces)

Write ```f (arg)``` and not ```f(arg)```.

#### (Position of ```&```)

Write ```const& v``` and not ```const &v```.

#### (No Whitespace before ```;```)

Write ```;``` and not ``` ;```

#### (Never omit braces)

Write ```if (cond) { body; }``` and not ```if (cond) body;```
